## Color Application Map

### Hero Section
```
Background: Gradient (Slate 50 → Blue 50 → White)
Animated Orbs: Electric Cyan 20%, Royal Blue 15%
Badge: 
  - Background: Cyan/Blue gradient 10% opacity
  - Border: Electric Cyan 30%
  - Text: Royal Blue → Deep Navy gradient
Headline: Slate 900
Gradient Text: Cyan → Royal Blue → Deep Navy
Subheadline: Slate 600
CTAs: Gradient primary (Cyan → Royal Blue)
Stats: 
  - Icons: Electric Cyan
  - Numbers: Slate 900
  - Text: Slate 600
```

### Services Section
```
Background: White
Heading: Slate 900
Description: Slate 600
Cards:
  - Background: Slate 50
  - Border: Slate 200
  - Icon background: Service-specific gradient
  - Hover: Electric Cyan glow
```

### Portfolio Section
```
Background: Gradient (White → Slate 50)

### Process Section
```

```

### Testimonials Section
```
Background: Slate 50

### Footer

## Spacing Guide (8px base unit)

### Vertical Spacing
```
Between sections: 96px (12 units)
Section padding: 96px top/bottom
Heading to content: 64px (8 units)
Content blocks: 48px (6 units)
Paragraphs: 24px (3 units)
List items: 16px (2 units)
```

### Horizontal Spacing
```
Container padding: 32px (4 units)
Card padding: 32px (4 units)
Button padding: 16px×40px
Grid gaps: 24px (3 units)
Icon to text: 12px (1.5 units)
```

### Component Spacing
```
Button groups: 16px gap
Form fields: 24px gap
Card grids: 32px gap
Navigation items: 32px gap
```

## Interactive States

### Buttons
```
Default: Gradient, shadow
Hover: Scale 1.05, brightness +10%, lift 2px
Active: Scale 0.98, shadow reduced
Focus: 3px cyan outline, 2px offset
Disabled: 50% opacity, no pointer
```

### Cards
```
Default: White, subtle shadow
Hover: Lift 4px, shadow increased, cyan glow
Active: Lift 2px
Focus: Cyan outline
```

### Links
```
Default: Royal Blue
Hover: Electric Cyan, underline
Active: Deep Navy
Visited: Purple
Focus: Cyan outline
```

### Form Inputs
```
Default: Slate 200 border
Focus: Royal Blue border, cyan glow
Error: Red border, error message
Success: Green border, checkmark
Disabled: Slate 100 background
```

## Micro-interactions

### On Scroll
- Fade in elements as they enter viewport
- Stagger animations (100ms delay between items)
- Parallax on hero background orbs
- Navigation shadow appears on scroll

### On Hover
- Card lift and glow
- Button scale and brighten
- Link color change
- Icon rotation/bounce

### On Click
- Button press animation
- Ripple effect on touch (mobile)
- Smooth scroll to section
- Modal fade in

### On Load
- Hero content cascades in (badge → headline → subheadline → CTAs → stats)
- Orbs float into position
- Navigation slides down
- Progress bars animate to value

```css
@media print {
  /* Hide non-essential elements */
  nav, footer, .scroll-indicator { display: none; }
  
  /* Optimize for print */
  body { background: white; color: black; }
  
  /* Show all content */
  .hidden-print { display: none; }
  
  /* Page breaks */
  section { page-break-before: always; }
  
  /* No animations */
  * { animation: none !important; }
}
```
